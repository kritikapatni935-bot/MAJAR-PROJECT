DESCRIPTION

The Advanced Quiz Game is a console-based application developed in the C programming language.  
It is designed to test a user’s knowledge through multiple-choice questions that are loaded from an external file. The program demonstrates the use of file handling, structures, dynamic memory allocation, modular programming, arrays, strings, randomization, and conditional logic.
To Play Quiz:
INSTRUCTION:
Press 1 and Enter

Type your name

Select category (or 0 for all)

Select number of questions

For each question, type the answer:
1, 2, 3, or 4

You can use:

50-50 lifeline (removes 2 wrong options)

Skip lifeline (skip one question)

After the quiz, your score will be shown.
